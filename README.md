
  A React app to pick colors from images.
  
  # Features
  
  - Preview color on hover
  - Copy/remove colors
  - Create Scheme
  - Zoom image up to 3x
  - Add an image via URL or upload your own image from your device
  - Edit Scheme
  - Import/export Schemes
  - Responsive
  
  ## Instructions
  
  Click on the help button on the left sidebar of the app.
